# swarmbook

和我一起探索去中心化网络的奥秘 - Rust libp2p 学习笔记与实战

## 项目结构

```
swarmbook/
├── examples/     # Rust libp2p 示例代码
├── docs/         # Astro + Starlight 教程站点
└── README.md
```

## 内容规划

- libp2p 核心概念
- Swarm 与网络行为
- 传输层与协议
- 实战项目

## 本地开发

```bash
# 运行示例
cd examples
cargo run --example <name>

# 启动文档站点
cd docs
npm run dev
```
